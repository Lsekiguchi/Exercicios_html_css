Principais formatações

Negrito/Destaque

Nessa frase temos um termo em negrito usando a tag <b> (não semântica)

Nessa outra frase, temos um termo em destaque usando a tag <strong> (semântica)

Itálico/Ênfase

Nessa frase, temos um termo em itálico usando a tag <i> (não semântica)

Nessa outra frase, temos um termo em ênfase usando a tag <em> (semântica)

Texto marcado

Podemos criar um texto marcado usando a tag <mark>

Para que a cor padrão do marca-texto mude, use css para alterá-la no head. Como está feito nesse arquivo.

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno nesse parágrafo

Texto riscado

Podemos marcar um texto como riscado para indicar que ele deve ser lido, mas não considerado. Para isso, usa-se a tag <del>

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois. Usa-se então a tag <ins>

Existe também o sublinhado com a tag <u> (não semântica)

Texto sobrescrito

Para inserir coisas do tipo: x2 - 1 = 0. Use a tag <sup>

Texto subescrito

Para inserir coisas do tipo: H20. Use a tag <sub>
